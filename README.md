# Análise de Carteira de Crédito Estruturado

## Objetivo
Analisar a qualidade de uma carteira de recebíveis, avaliando distribuição, retorno e risco.

## Base de Dados
~346 mil registros
Período: 2022–2025
Variáveis: valor, taxa, pagamentos

## Metodologia
Tratamento de dados (datas e nulos)
Cálculo de taxa média ponderada
Cálculo de inadimplência (cash)
Análise por tempo e convênio

## Principais Insights

Inadimplência total de ~11%
Deterioração da carteira a partir de 2025 (~35%)
Concentração em convênios de maior risco

## Conclusão:
O risco não foi corretamente precificado.
